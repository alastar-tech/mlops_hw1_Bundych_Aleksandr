### Цель проекта:
создание воспроизводимого ML-pipeline с использованием DVC и MLflow


### Описание пайплайна: состоит аз двух шагов
- подготовка данных (prepare.py)
- обучение модели линейной регрессии (train.py)


### Как запустить:
через командную строку
1.  **Склонируйте репозиторий:**
    git clone https://github.com/alastar-tech/mlops_hw1_Bundych_Aleksandr.git
    cd mlops_hw1_Bundych_Aleksandr

2.  **Установите зависимости:**
    pip install -r requirements.txt

3.  **Загрузите данные из DVC хранилища:**
    dvc pull (для доступа, как я понял, нужчно чтобы владелец папку (я) указал почту того, кто подключается. Авторизация по поему аккаунту работает, а более простой вариант с dvc studio не поулчился т.к. лк не активировался)

4.  **Воспроизведите пайплайн:**
    dvc repro



### Где смотреть UI MLflow: 
**для запуска интерфейсf MLflow**:
- команда  в терминале: mlflow ui
- после запуска откройте в браузере адрес `http://127.0.0.1:5000`
